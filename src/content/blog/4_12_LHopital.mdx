## Introducción

La **regla de L'Hôpital** es una técnica que permite calcular ciertos límites de funciones que, a primera vista, son difíciles de resolver debido a que generan una indeterminación. Esta regla es útil cuando encontramos fracciones cuyo numerador y denominador tienden a valores como **0** o **infinito**, lo que hace que no se pueda determinar el valor del límite de manera directa.

Por ejemplo, si al evaluar un límite encontramos algo como **0 entre 0** o **infinito entre infinito**, estamos ante una indeterminación, y la regla de L'Hôpital es una herramienta efectiva para abordar este tipo de problemas.

---

## ¿En qué consiste la regla de L'Hôpital?

La idea detrás de la regla de L'Hôpital es analizar cómo cambian el numerador y el denominador de una fracción mientras se acerca al valor que genera la indeterminación. Para hacer esto:

1. Se calcula la derivada del numerador.
2. Se calcula la derivada del denominador.
3. Se vuelve a evaluar el límite utilizando las derivadas en lugar de las funciones originales.

Si el nuevo límite sigue siendo una indeterminación, se repite el proceso hasta obtener un resultado claro.

---

## ¿Cuándo se aplica?

La regla de L'Hôpital se puede aplicar en dos situaciones específicas:

1. Cuando el límite directo de una fracción produce **0/0**.
2. Cuando el límite directo produce **infinito/infinito**.

**Nota importante:** No puede aplicarse si el límite no genera una indeterminación. Además, para que esta regla sea válida, el numerador y el denominador deben ser derivables cerca del punto en cuestión.

---

## Ejemplo práctico: Cálculo de un límite

Supongamos que queremos calcular el límite de la fracción:

(seno de x) / x

Cuando x se aproxima a 0.

1. **Evaluación directa:**  
   - El seno de 0 es igual a 0, por lo que el numerador es 0.
   - El denominador, x, también es 0.

   Esto genera una indeterminación de tipo **0/0**, por lo que podemos aplicar la regla de L'Hôpital.

2. **Aplicar la regla:**  
   - Derivamos el numerador, seno de x, cuya derivada es coseno de x.
   - Derivamos el denominador, x, cuya derivada es 1.

   Ahora tenemos una nueva fracción: (coseno de x) / 1.

3. **Evaluación del nuevo límite:**  
   - Cuando x se aproxima a 0, el coseno de x se aproxima a 1.

Por lo tanto, el límite de (seno de x) / x cuando x se acerca a 0 es igual a **1**.

---

## Ejemplo adicional: Exponenciales

Calculemos el límite de la fracción:

(e^x - 1) / x

Cuando x tiende a 0.

1. **Evaluación directa:**  
   - Al evaluar directamente, obtenemos 0/0, que es una indeterminación.

2. **Aplicamos la regla de L'Hôpital:**  
   - Derivamos el numerador, e^x - 1, cuya derivada es e^x.
   - Derivamos el denominador, x, cuya derivada es 1.

   Ahora la fracción es: (e^x) / 1.

3. **Evaluación del nuevo límite:**  
   - Cuando x se aproxima a 0, e^x se aproxima a 1.

Por lo tanto, el límite de (e^x - 1) / x cuando x se aproxima a 0 es igual a **1**.

## Ejercicios

1. Calcula el límite de (ln(x)) / x cuando x tiende a infinito.
2. Encuentra el límite de (x² - 4) / (x - 2) cuando x se aproxima a 2.
3. Resuelve el límite de (tan(x)) / x cuando x se aproxima a 0.

---

## Respuestas

1. El límite de (ln(x)) / x cuando x tiende a infinito:
   - Evaluación directa: ln(x) e x ambos tienden a infinito, generando una indeterminación.
   - Aplicamos L'Hôpital: Derivada del numerador es 1/x, derivada del denominador es 1.
   - Nuevo límite: (1/x) / 1 = 1/x. Al evaluar en infinito, el resultado es **0**.

2. El límite de (x² - 4) / (x - 2) cuando x se aproxima a 2:
   - Evaluación directa: 0/0, indeterminación.
   - Aplicamos L'Hôpital: Derivada del numerador es 2x, derivada del denominador es 1.
   - Nuevo límite: 2(2) / 1 = **4**.

3. El límite de (tan(x)) / x cuando x se aproxima a 0:
   - Evaluación directa: 0/0, indeterminación.
   - Aplicamos L'Hôpital: Derivada del numerador es sec²(x), derivada del denominador es 1.
   - Nuevo límite: sec²(0) / 1 = 1. Por lo tanto, el resultado es **1**.

---

## Conclusión

La regla de L'Hôpital es una herramienta esencial en el cálculo de límites cuando se encuentran indeterminaciones. Al reemplazar el numerador y el denominador por sus respectivas derivadas, el proceso se simplifica significativamente, permitiendo resolver problemas que inicialmente parecen complicados. Este método no solo es práctico, sino también efectivo para comprender cómo las funciones cambian en relación unas con otras.
