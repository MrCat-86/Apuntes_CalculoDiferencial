## Introducción

Las **operaciones con funciones** son procesos matemáticos mediante los cuales combinamos o manipulamos dos o más funciones para obtener una nueva función. Las operaciones más comunes son la **suma**, la **resta**, la **multiplicación** y la **división** de funciones. A continuación, exploraremos cada una de ellas con ejemplos prácticos.

---

## 1. Suma de Funciones

### Definición:
Si tenemos dos funciones \(f(x)\) y \(g(x)\), la suma de funciones se define como la función que resulta de sumar los valores de \(f(x)\) y \(g(x)\) para un mismo valor de \(x\). Esto se denota como:

**(f + g)(x) = f(x) + g(x)**

### Ejemplo:
Supongamos que tenemos las siguientes funciones:

- \( f(x) = 2x + 1 \)
- \( g(x) = x^2 \)

La suma de estas dos funciones sería:

**(f + g)(x) = (2x + 1) + x^2**

Esto nos da la nueva función:

**f(x) + g(x) = x^2 + 2x + 1**

### Propiedades de la Suma de Funciones:
- La suma de dos funciones es conmutativa: **f + g = g + f**
- La suma de dos funciones es asociativa: **(f + g) + h = f + (g + h)**

---

## 2. Resta de Funciones

### Definición:
La resta de dos funciones se obtiene restando los valores de \(f(x)\) y \(g(x)\) para un mismo valor de \(x\). Esto se denota como:

**(f - g)(x) = f(x) - g(x)**

### Ejemplo:
Supongamos que tenemos:

- \( f(x) = 3x + 2 \)
- \( g(x) = x^2 \)

La resta de estas dos funciones sería:

**(f - g)(x) = (3x + 2) - x^2**

Esto nos da la nueva función:

**f(x) - g(x) = -x^2 + 3x + 2**

### Propiedades de la Resta de Funciones:
- La resta de funciones no es conmutativa: **f - g ≠ g - f**
- La resta de funciones es asociativa: **(f - g) - h = f - (g + h)**

---

## 3. Multiplicación de Funciones

### Definición:
La multiplicación de dos funciones se obtiene multiplicando los valores de \(f(x)\) y \(g(x)\) para un mismo valor de \(x\). Esto se denota como:

**(f * g)(x) = f(x) * g(x)**

### Ejemplo:
Supongamos que tenemos:

- \( f(x) = 3x \)
- \( g(x) = x + 1 \)

La multiplicación de estas dos funciones sería:

**(f * g)(x) = (3x) * (x + 1)**

Esto nos da la nueva función:

**f(x) * g(x) = 3x² + 3x**

### Propiedades de la Multiplicación de Funciones:
- La multiplicación de funciones es conmutativa: **f * g = g * f**
- La multiplicación de funciones es asociativa: **(f * g) * h = f * (g * h)**

---

## 4. División de Funciones

### Definición:
La división de dos funciones se obtiene dividiendo los valores de \(f(x)\) y \(g(x)\) para un mismo valor de \(x\). Esto se denota como:

**(f / g)(x) = f(x) / g(x)**, siempre y cuando **g(x) ≠ 0**

### Ejemplo:
Supongamos que tenemos:

- \( f(x) = x^2 + 1 \)
- \( g(x) = x + 1 \)

La división de estas dos funciones sería:

**(f / g)(x) = (x² + 1) / (x + 1)**

Esto nos da la nueva función:

**f(x) / g(x) = (x² + 1) / (x + 1)**

### Propiedades de la División de Funciones:
- La división de funciones no es conmutativa: **f / g ≠ g / f**
- La división de funciones no es asociativa: **(f / g) / h ≠ f / (g / h)**

## Ejercicios

### Ejercicio 1: Suma de Funciones

Dadas las siguientes funciones:

- \( f(x) = 4x - 3 \)
- \( g(x) = x^2 + 2x \)

Encuentra la suma de \( f(x) \) y \( g(x) \).

**Solución:**

La suma es:

**(f + g)(x) = (4x - 3) + (x^2 + 2x) = x^2 + 6x - 3**

### Ejercicio 2: Resta de Funciones

Dadas las siguientes funciones:

- \( f(x) = 5x + 4 \)
- \( g(x) = x^2 + 3x \)

Encuentra la resta de \( f(x) \) y \( g(x) \).

**Solución:**

La resta es:

**(f - g)(x) = (5x + 4) - (x^2 + 3x) = -x^2 + 2x + 4**

### Ejercicio 3: Multiplicación de Funciones

Dadas las siguientes funciones:

- \( f(x) = x + 1 \)
- \( g(x) = 2x \)

Encuentra la multiplicación de \( f(x) \) y \( g(x) \).

**Solución:**

La multiplicación es:

**(f * g)(x) = (x + 1) * (2x) = 2x² + 2x**

### Ejercicio 4: División de Funciones

Dadas las siguientes funciones:

- \( f(x) = x^2 + 3x + 2 \)
- \( g(x) = x + 1 \)

Encuentra la división de \( f(x) \) y \( g(x) \).

**Solución:**

La división es:

**(f / g)(x) = (x² + 3x + 2) / (x + 1)**

Realizando la división de polinomios, obtenemos:

**(f / g)(x) = x + 2**
