## Introducción

La programación es una herramienta fundamental que nos permite interactuar con las computadoras a través de instrucciones específicas. En esta sección, abordamos conceptos básicos que todo programador debe conocer.

## Conceptos

1. **Lenguaje de Programación**: Es un sistema formal para escribir instrucciones que las computadoras pueden interpretar y ejecutar.
2. **Compilación e Interpretación**: Los programas deben ser traducidos a lenguaje máquina mediante un compilador o un intérprete.
3. **Sintaxis y Semántica**: 
   - **Sintaxis**: Conjunto de reglas que dictan cómo escribir correctamente el código.
   - **Semántica**: El significado lógico y funcional detrás de cada instrucción.

```java
// Ejemplo de sintaxis en Java
public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("¡Hola, mundo!");
    }
}
```
