## Introducción

Los lenguajes de programación permiten representar y manipular información en la computadora mediante diversos elementos del lenguaje. Estos elementos incluyen los **tipos de datos**, **literales**, **constantes**, **variables**, **identificadores**, **parámetros**, **operadores** y la **salida de datos**. Todos estos componentes son fundamentales para poder escribir y ejecutar un programa.

## Tipos de Datos

Los **tipos de datos** definen qué tipo de valor puede almacenar una variable y qué operaciones se pueden realizar con él. Los tipos de datos más comunes incluyen:

- **Enteros** (`int`, `long`): Números enteros sin decimales.
- **Flotantes** (`float`, `double`): Números con decimales.
- **Booleanos** (`boolean`): Valores de verdadero o falso.
- **Caracteres** (`char`): Un solo carácter.
- **Cadenas de texto** (`String`): Una secuencia de caracteres.

### Ejemplo en Java:

```java
int edad = 25;            // Entero
double temperatura = 36.6; // Flotante
boolean esAdulto = true;  // Booleano
char inicial = 'A';       // Carácter
String nombre = "Juan";   // Cadena de texto
```

## Literales

Los **literales** son valores constantes que se asignan a las variables en un programa. Pueden ser de diferentes tipos, como números, texto o valores booleanos.

### Ejemplos:

- **Enteros**: `25`
- **Flotantes**: `36.6`
- **Cadenas de texto**: `"Hola Mundo"`
- **Booleanos**: `true` o `false`
- **Caracteres**: `'A'`

### Ejemplo en Java:

```java
int numero = 10;  // Literal entero
double valor = 9.99;  // Literal flotante
boolean activo = false; // Literal booleano
char letra = 'X';  // Literal de carácter
String mensaje = "Bienvenido"; // Literal de cadena de texto
```

## Constantes

Las **constantes** son valores que no cambian durante la ejecución de un programa. Se definen una sola vez y mantienen su valor a lo largo del programa. Se utilizan para almacenar valores que no deberían ser modificados.

### Ejemplo en Java:

```java
final double PI = 3.14159;  // Constante
final int MAX_EDAD = 100;   // Constante
```

## Variables

Las **variables** son espacios de almacenamiento en la memoria que pueden contener valores que cambian durante la ejecución del programa. Se deben declarar antes de usarlas y asignarles un tipo de dato.

### Ejemplo en Java:

```java
int num1 = 10;  // Declaración y asignación
num1 = 20;      // Modificación del valor de la variable
```

## Identificadores

Los **identificadores** son nombres que se le asignan a las variables, constantes, funciones y otros elementos del programa. Deben comenzar con una letra, un guion bajo (`_`) o un signo de dólar (`$`) y pueden contener números, pero no pueden empezar con un número.

### Reglas para identificadores:

- No deben contener espacios.
- No deben ser palabras reservadas del lenguaje.
- Deben ser descriptivos, para que el código sea fácil de entender.

### Ejemplo en Java:

```java
int edadUsuario = 25;  // Buen identificador
String nombreCompleto = "Juan Pérez";  // Buen identificador
```

## Parámetros

Los **parámetros** son variables que se pasan a las funciones o métodos cuando se llaman. Los parámetros permiten que las funciones trabajen con diferentes valores cada vez que se llaman.

### Ejemplo en Java:

```java
public class Suma {
    public static int sumar(int num1, int num2) {  // Parámetros
        return num1 + num2;
    }

    public static void main(String[] args) {
        int resultado = sumar(5, 7);  // Paso de argumentos
        System.out.println("Resultado: " + resultado);
    }
}
```

## Operadores

Los **operadores** son símbolos que realizan operaciones sobre variables y valores. Los principales tipos de operadores son:

- **Aritméticos**: `+`, `-`, `*`, `/`, `%`
- **Relacionales**: `==`, `!=`, `<`, `>`, `<=`, `>=`
- **Lógicos**: `&&` (AND), `||` (OR), `!` (NOT)
- **De asignación**: `=`, `+=`, `-=`, `*=`, `/=`

### Ejemplo en Java:

```java
int a = 5, b = 10;
int suma = a + b;  // Operador aritmético
boolean esMayor = a > b;  // Operador relacional
boolean resultadoLogico = (a < b) && (b > 5);  // Operador lógico
a += 2;  // Operador de asignación (equivalente a a = a + 2)
```

## Salida de Datos

La **salida de datos** se refiere a la forma en que un programa muestra la información al usuario. En Java, la salida de datos generalmente se realiza a través de `System.out.println()`.

### Ejemplo en Java:

```java
public class Salida {
    public static void main(String[] args) {
        String mensaje = "Hola Mundo";
        System.out.println(mensaje);  // Salida de datos
    }
}
```
